# 🎯 Face Recognition System

A Python-based Face Recognition System that detects and recognizes human faces using computer vision and machine learning techniques.

---

## 📌 Features

✅ Real-time face detection
✅ Face recognition using trained dataset
✅ Easy to use interface
✅ Fast and accurate recognition
✅ Can be extended for attendance/security systems

---

## 🛠️ Technologies Used

* Python
* OpenCV
* NumPy
* Face Recognition Library

---

## 📂 Project Structure

```
face-recognition/
│
├── dataset/              # Training images
├── trainer/              # Trained model files
├── face_recognition.py   # Main recognition script
├── train_model.py        # Model training script
├── requirements.txt      # Dependencies
└── README.md
```

---

## ⚙️ Installation

### 1️⃣ Clone the repository

```
git clone https://github.com/Shreeram-ux/face-recognition.git
```

### 2️⃣ Navigate to project folder

```
cd face-recognition
```

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

---

## ▶️ How to Run

### Step 1 — Train the model

```
python train_model.py
```

### Step 2 — Start face recognition

```
python face_recognition.py
```

---

## 📸 Dataset

Add images of persons inside the `dataset` folder.
Folder name should be the person’s name.

Example:

```
dataset/
   ├── John/
   │     img1.jpg
   │     img2.jpg
   ├── Smith/
         img1.jpg
```
